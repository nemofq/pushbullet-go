# Privacy Policy for Push Go

**Last Updated: July 17, 2026**

## Data Collection

Push Go does not collect, store, or transmit any personal data from users. This extension operates solely as a client interface for the Pushbullet API.

## Data Usage

This extension:
- Communicates exclusively with Pushbullet servers (API and content delivery domains) to send, receive, and display pushes
- Displays content received from Pushbullet servers
- Transmits only the content you explicitly choose to send to the Pushbullet API

## Local Storage

The extension stores the following data in your browser:

1. Your Pushbullet access token, and, if you enable end-to-end encryption, a locally derived encryption key (your encryption password itself is never stored)
2. Account, device, and chat information retrieved from the Pushbullet API
3. Extension preferences and settings
4. Recent push and mirrored notification history

This data is not transmitted to any servers other than the Pushbullet API. If you have Chrome sync enabled, some of it may be synced across your browsers by Chrome itself.

## Third-Party Services

This extension integrates with the Pushbullet API service. All data handling by Pushbullet is governed by their privacy policy, which you can review at: https://www.pushbullet.com/privacy

## Contact

For questions about this privacy policy, please create an issue on the project repository.
